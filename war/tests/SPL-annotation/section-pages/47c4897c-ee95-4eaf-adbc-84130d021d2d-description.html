DESCRIPTION

Clozapine, an atypical antipsychotic drug, is a tricyclic dibenzodiazepine derivative. The chemical name of clozapine is 8-chloro-11-(4-methyl-1-piperazinyl)-5H-dibenzo [b,e] [1,4] diazepine and it has the following structural formula:

C18H19ClN4 M.W. 326.83
Clozapine is a yellow, crystalline powder, very slightly soluble in water. Each tablet, for oral administration, contains 25 mg, 50 mg, 100 mg, or 200 mg clozapine. In addition, each tablet contains the following inactive ingredients: colloidal silicon dioxide, corn starch, magnesium stearate, povidone, pregelatinized starch, and talc. Additionally, the 25 mg and 100 mg tablets contain lactose monohydrate and the 50 mg and 200 mg tablets contain anhydrous lactose and crospovidone.
